# Install Fish on Mac 

## Install Homebrew
https://brew.sh/
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

```

## Install and customize fish

```
Install using installer

List shells
cat /etc/shells

Switch shell
chsh -s /usr/local/bin/fish

check the current shell
echo $SHELL
```
